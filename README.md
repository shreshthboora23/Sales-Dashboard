# Sales-Dashboard
# Ecommerce Sales Dashboard Power BI Project
# Created interactive dashboard to track and analyze online sales data.

This project presents an interactive Ecommerce Sales Dashboard built using Power BI to analyze sales performance, profit trends, and customer behavior across different dimensions. The dataset was initially taken in CSV format, which was cleaned and transformed before visualization.

1. Data Preparation

* Imported raw data from CSV file
* Performed data cleaning (handled missing values, removed duplicates, formatted columns)
* Created calculated measures for Sales, Profit, Quantity, and Average Order Value.

2. Dashboard Overview

  Top KPIs

* Total Sales (438K) – Overall revenue generated
* Total Quantity (5615) – Total units sold
* Total Profit (37K) – Net profit earned
* Average Order Value (121K) – Average revenue per order

3. Charts & Insights

* Profit by Month (Bar Chart)
  Shows monthly profit trends to identify peak and low-performing months.

* Profit by Sub-Category (Horizontal Bar Chart)
  Highlights which product sub-categories (like Printers, Bookcases, Saree, etc.) contribute most to profit.

* Profit by Customer/Name (Bar Chart)
  Displays profit contribution by individual customers, helping identify high-value customers.

* Quantity by Category (Donut Chart)
  Breaks down product categories (Clothing, Electronics, Furniture) based on quantity sold.

* Sum of Amount by State (Bar Chart)
  Represents state-wise sales distribution, showing top-performing regions like Maharashtra and Madhya Pradesh.

* Quantity by Payment Mode (Donut Chart)
  Shows how customers prefer to pay (UPI, Debit Card, Credit Card, EMI).

---

4. Filters & Interactivity

* Quarter Filter (Q1–Q4)
* Month Filter
  These filters allow dynamic exploration of data for better decision-making.
